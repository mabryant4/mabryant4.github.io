Graphics: Moon Phase Diagram

This project represents a moon phase diagram from November 2020 to November 2021. It uses WebGL, JavaScript, and HTML to model a sphere mapped with the texture of the moon and rotate the lighting to represent the different moon phases. This diagram is for the city of Tuscaloosa, so in other cities there will likely be some differences. For more information on how to run this project see the read me file in the GitHub repository.

Artificial Intelligence: Poker Player

This project emulates a game of 5 card draw poker without betting. Each player is randomly distributed a hand of cards from a standard deck and must decide which cards to replace, in hopes of achieving the best hand. The player I created is the class MarleeBryant. For more information on the gameplay, strategy, and performance, see the read me file in the GitHub repository.
